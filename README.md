# rain-prediction-research-paper
the code of the neural network development for the assignment in the AI lecture at DHBW stuttgart in winter semester of 2025/26

only the code of the neural network model is uploaded here. to run the model the dataset used has to be downloaded from kaggle and the python skript has to be in the same directory as the .csv of the dataset. the dataset file should also not be renamed.

dataset: https://www.kaggle.com/datasets/jsphyg/weather-dataset-rattle-package
